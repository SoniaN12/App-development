# SOUL SHOP
<img width="1416" height="693" alt="Screen Shot 2026-03-22 at 1 23 56 PM" src="https://github.com/user-attachments/assets/9309492f-1ebb-4c7d-a931-cdfcca56235c" />

Soul Shop is a web application  that simulates a real world online store where users can browse products, add new items , search for products , and simulate purchasing through a payment gateway.
The Soul Shop application is designed to provide a smooth and engaging shopping experience, it is hence divided into several key sections that reflect ecommerce functionality.
